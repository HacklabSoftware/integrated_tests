# integration

## Run with bun

To install bun follow the instructions at [bun.sh](https://bun.sh)

To install dependencies using bun

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## Run with docker

To build docker

```bash
docker build -t test-script .
```

To run docker

```bash
docker run test-script
```
**One can change configurations in config.s file**
